# Real-time Chat Application - Frontend Repository

### Social & Messaging Platform – Frontend Developer

This repository contains the Client-Side application for a real-time messaging platform. The application is highly optimized, features a beautiful user interface with dynamic Light/Dark Mode integration, and fully supports instantaneous real-time updates.


### ⚙️ Local Setup Instructions

1. Install project dependencies:
   ```bash
   npm install
   ```
2. Create or update environment configuration files (e.g., `.env`) to point the `VITE_API_URL` to your local Backend API.
3. Start the Vite development environment:
   ```bash
   npm run dev
   ```
4. Access the frontend client via browser at `http://localhost:5173`.
